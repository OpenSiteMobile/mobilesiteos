MobileSiteOS™ JavaScript Demo

Our MobileSiteOS™ JavaScript "framework" is basically an assortment of function specific modules, organized underneath a mobile optimized interface. We define it as a mobile aware code layer between some of the best standard JavaScript frameworks (Modernizr, jQuery, jQuery-UI, Underscore.js, Backbone.js, Hammer.js, Bootstrap) and standard HTML5 web pages.

One of the primary features of MobileSiteOS™ is the ability to load specific page content "on demand", including modules, widgets and plug-ins for that page's html code. Plus, MobileSiteOS™ provides a mobile aware environment, for optimized page display through numerous RWD techniques and code. This is a "single page" website example.

jQuery UI Integration
This section includes examples of jQuery-UI integration into MobileSiteOS™. We "on demand" load all jQuery-UI widget modules and CSS, (excluding core code and CSS), so only the required code and it's specific CSS are loaded for each widget or function utilized.
Bootstrap Integration
This section includes examples of Bootstrap UI integration into MobileSiteOS™. Similar to jQuery-UI, we "on demand" load many of the Bootstrap UI widgets as modules so only that code and it's specific css, are loaded. Other more universally utilized modules are loaded based on HTML DOM inspection via jQuery.
