Our Goal at OpenSiteMobile

We are developers for the mobile web, using only open source software technologies. Our goal is to aid in the creation of "responsive display websites". We do this through the integration of the best available web technologies and internet standard practices.

Our Software

We call our software MobileSiteOS™. Mobile is now dominant, but the desktop and other display sizes will never go away. Thus MobileSiteOS™ software is an "operational layer" between many standard JavaScript functional libraries and the final web page. Our "layer" tries to provide for responsive web pages through the thoughtful integration of many current and developing HTML5, JavaScript and CSS technologies.

Originally, this concept was called the "One Web". Now it is more typically call "responsive design" or "adaptive design". As it turns out, this is not a simple task as the multitude of semi-responsive websites currently littering the web can attest to. By utilizing specifically sized display adjusted pages, MobileSiteOS™ can allow almost any device access to your website with pleasing and controlled results.

Software Dependencies

We have made extensive use of the following JavaScript libraries, which form the base foundation for our software. jQuery, jQuery-UI and BackboneJS are called as separate files. UnderscoreJS is include in our 'base' file, as well as the Modernizr tests utilized by MobileSiteOS. These frameworks are all loaded at initial page load, and thus are available throughout later content injection. It is important to note that MobileSiteOS features dynamic module loading at any point in the content rendering process, so any modular code requiring these dependencies will still function as expected.
  • jQuery
  • jQuery-UI
  • Modernizr (Incl'd)
  • UnderscoreJS (Incl'd)
  • BackboneJS
  • Bootstrap (opt'l)
  • AngularJS (opt'l)
