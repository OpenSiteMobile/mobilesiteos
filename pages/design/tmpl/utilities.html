Mobile Utilities and Services

Here are several links to on-line site transcoders for mobile devices. All you do to use a transcoder is go to their website and enter a url into the site's home page input box. Then the transcoder will take the returned web page code for that url and render it in a format more suitable for mobile display. It's not perfect, but can sometimes do a reasonable job of "mobilizing" a "desktop only" website.

Many (most?) transcoder sites allow you to encode the "to be rendered" url into a url query to their site, thus making it possible to pass rendering directly through from your website or app without having to have the user enter url text directly. Still, the problem is that you don't know what the user's device is ahead of time, and how it will actually render pages of the linked site.
