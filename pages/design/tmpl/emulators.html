Below is a list of several on-line mobile device emulators. Most of these emulators can be used to test how your mobile compliant website will look on different mobile devices. They allow the input of url's which then render your pages inside the emulated small screen device.

Are these emulators perfect? No, but they are a good way to get a basic idea of what page code from your website works and what doesn't. The main drawback when testing a MobileSiteOSâ„¢ installation is that many don't accurately mask the actual screen size or window size to JavaScript. Thus, our use of JavaScript to determine display size may be incorrect because it is getting the current page viewing size and not an emulated device size. You will have to set it manually in order to see the appropriate size for each emulated device. Reducing the browser window size to just the size of the emulator may work, but might also obscure other emulator interactions.
